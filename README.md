# 🎮 Pygame Menu Asset
Un système de menu responsive, moderne et facile à intégrer pour vos projets de jeux vidéo avec Pygame. Ne perdez plus de temps à coder l'interface, concentrez-vous sur le gameplay !

# 🚀 Fonctionnalités
**Responsive Design :** Le menu s'adapte automatiquement à la taille de votre fenêtre.

**Interactivité complète :** Gestion du survol (hover) et des clics avec retour visuel.

**Prêt à l'emploi :** Structure modulaire permettant d'ajouter des boutons en quelques lignes.

# 🛠️ Installation
- Clonez le projet :
```Bash
git clone https://github.com/Tortue-Enjoye/MenuAsset.git
cd MenuAsset
```
- Installez Pygame (si ce n'est pas déjà fait) :
```Bash
pip install pygame
```

- Lancez la démo :
```Bash
python main.py
```
